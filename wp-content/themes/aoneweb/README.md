# DC Skeleton
A customizable starter theme for Wordpress.

## Install and use
- Node version 14
- Pull this repo and run `npm install`<br>
- For watch: `npm run watch`<br>
- For dev: `npm run dev`<br>
- For production: `npm run prod`<br>
- Also remember to change the .env file.
